モバイル「ホームに追加」用アイコンについて
========================================

次の PNG をこのフォルダ（public/）に置いてください。

・ icon-192.png … 192×192 ピクセル（Android など）
・ icon-512.png … 512×512 ピクセル（Android など）

※ Apple 端末は icon-192.png をホーム画面アイコンに使います。

作り方の例：
1. public/favicon.svg をブラウザで開き、スクリーンショットで 512×512 を保存 → icon-512.png
2. 画像編集ソフトで 192×192 にリサイズ → icon-192.png
3. または https://realfavicongenerator.net/ などで favicon.svg から一括生成

アイコンを置いたあと、ビルドして公開用に push すると反映されます。
